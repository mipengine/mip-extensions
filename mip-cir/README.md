# mip-cir 

mip-cir 用来显示百度智荐推荐内容

描述|百度智荐通用组件
----|----
类型| 通用
支持布局|N/S
所需脚本|https://mipcache.bdstatic.com/static/v1/mip-cir/mip-cir.js

## 示例

```html
    <mip-cir tid="123456" read=".article"></mip-cir>
    <div>内容</div>
```

## 属性

### tid

说明：推荐位id
必填：是
格式：数字
单位：无

### read

说明：文章阅读深度统计dom元素容器的选择器
必填：是
格式：CSS选择器
单位：无
