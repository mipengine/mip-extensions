# mip-ls-pagination

mip-ls-pagination 用来实现招聘信息列表的分页功能

标题|内容
----|----
类型|业务
支持布局|N/S
所需脚本|https://c.mipcdn.com/static/v1/mip-ls-pagination/mip-ls-pagination.js

## 示例
```
<mip-ls-pagination data-page-size="5">
    <ul>
        <li>1</li>
        <li>2</li>
        <li>3</li>
        <li>4</li>
        <li>5</li>
        <li>6</li>
        <li>7</li>
    </ul>
</mip-ls-pagination>
```

## 属性

### data-page-size

说明：每页显示的条数，（Number) 。默认每页显示5条记录。
必选项：否
