# mip-wl-swiper
mip-wl-swiper实现一个卡片轮播效果

##示例
<code>
                <mip-swiper class="swiper-container" id="team-recommend" direction="300" autoplay="true">
                <div class="swiper-wrapper">
                    <div class="swiper-slide"></div>
                    <div class="swiper-slide"></div>
                    <div class="swiper-slide"></div>
                </div>
            </mip-swiper>
</code>

##属性
<b>direction</b>
说明：过渡时间
必选项：否
类型：数字
取值范围： >0
单位：毫秒（ms）
默认值：350

<b>dealy</b>
说明：等待时间
必选项：否
类型：数字
取值范围： >1000
单位：毫秒（ms）
默认值：3000


##注意事项
无
