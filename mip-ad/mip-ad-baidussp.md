# 类型: ad-ssprelease ssp直投广告

## 支持布局

- reponsive
- fixed-height

## 示例

### 基本使用

```html
<mip-ad 
    type="ad-baidussp"
    sspId="xxx">
</mip-ad>
```

## 属性

### type

说明：广告类型  
必选项：是  
类型：字符串  
取值：ad-baidussp  
默认值：无

## 注意事项

SSP直投富媒体在满足以下3个条件

- 支持https
- 可以异步执行
- 嵌入广告（非悬浮）

以上条件，在网盟侧未做强制要求。
