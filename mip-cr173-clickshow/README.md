# mip-cr173-clickshow
mip-cr173-clickshow 用来支持下载详情页点击轮流显示内容

标题|内容
----|----
类型|业务
支持布局|N/S
所需脚本|https://mipcache.bdstatic.com/static/v1/mip-showtanceng/mip-cr173-clickshow.js

## 示例

```
<mip-cr173-hideshow>
	<div class="f-showhide-function" data-btnshow="true">
    	<b class="f-hideshow-btn"  data-num="0">点我</b>
        <div class="f-hideshow-cont">123</div>
        <div class="f-hideshow-cont">789</div>
    </div>    
</mip-cr173-hideshow>
```

# 属性

组件涉及的属性字段: 点击轮流显示内容,data-btnshow控制按钮是否显示
