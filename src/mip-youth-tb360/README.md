# mip-youth-tb360

mip-youth-tb360 青网移动适配页头部360广告

标题|内容
----|----
类型|通用，定制
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-youth-tb360/mip-youth-tb360.js

## 示例

移动适配页头部360第三方广告插件
```html
	
<mip-youth-tb360
     ayid="y74003"
	 adtype="ad-dsf"
     >
</mip-youth-tb360>
```

## 属性

### ayid

说明：扩展组件广告id  
必填：是
类型：字符串

### adtype

说明：广告类型  
必填：是
类型：字符串
取值:ad-dsf
