# mip-youth-ttyd

mip-youth-ttyd 青网移动适配页精彩推荐广告

标题|内容
----|----
类型|通用，定制
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本https://c.mipcdn.com/static/v1/mip-youth-ttyd/mip-youth-ttyd.js

## 示例

移动适配页精彩推荐第三方广告插件
```html
<div>
   <div id="toutiao_container"></div>
</div>	
<mip-youth-jctj 
     ayid="y74002"
	 adtype="ad-dsf"
     >
</mip-youth-jctj>
```

## 属性

### ayid

说明：扩展组件广告id  
必填：是
类型：字符串

### adtype

说明：广告类型  
必填：是
类型：字符串
取值:ad-dsf
