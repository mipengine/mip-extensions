# mip-audio 音频播放

提供了一个音频播放组件

标题|内容
----|----
类型|通用
支持布局| N/S
所需脚本|https://mipcache.bdstatic.com/static/v1/mip-audio/mip-audio.js

## 示例

``` html
<mip-audio 
    src="https://www.mipengine.org/static/RonaldJenkees-GuitarSound.mp3"
></mip-audio>
```

## 属性

### src

说明：音频地址  
必填：是  
格式：url  
使用限制：必须是https的
