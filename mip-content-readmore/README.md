## 属性

### element

说明：需要控制的元素，仅支持jquery的id和class元素选择器语法 
必选项：否  
类型：字符串
默认值：.content

### maxlen

说明：剔除空格、html标签后的文本长度（大约）
必选项：否  
类型：正整数
默认值：666

### buttitle

说明：按钮名称
必选项：否  
类型：字符串
默认值：查看更多
