# mip-hk-showarticle

显示文章更多详情

|标题|内容|
|---|---|
|类型|业务|
|支持布局|N/S|
|所需脚本|https://c.mipcdn.com/static/v1/mip-hk-showarticle/mip-hk-showarticle.js|

## 示例

在MIP HTML中,直接使用标签, 用于显示文章更多详情。

```
    <mip-hk-showarticle page="erji_detail_news" type="fengchao">
        <div id="J_show_art">
            <span>展开全文</span>
        </div>
    </mip-hk-showarticle>
```
## 属性

### page

说明：不同位置对应不同统计参数
必选项：否
类型：字符串

### type

说明：不同渠道显示文章长度不同
必选项：否
类型：字符串
