# mip-qbaobei-showmore

mip-qbaobei-showmore 用来支持页面显示更多文字

标题|内容
----|----
类型|业务
支持布局|N/S
所需脚本|http://mipcache.bdstatic.com/static/v1/mip-qbaobei-showmore/mip-qbaobei-showmore.js

## 示例

```html
<mip-qbaobei-showmore>
	<div class="video-wrap">
		<div class="introd">
			<div class="introd-title">简介：</div>
			<div class="introd-wrap threeline"><p>天空之城主题曲纯音乐是一首非常经典的<a href="http://m.qbaobei.com/a/286548.html" title="钢琴" class="innerlink" target="_blank">钢琴</a>曲，由日本著名音乐人、作曲家、钢琴家久石让创作演出。！</span></p></div>
			<p class="get-more"><span>全部<em class="arr"></em></span></p>
		</div>
	</div>
</mip-qbaobei-showmore>
```
