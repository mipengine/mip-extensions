# mip-bsml-fixedbar

mip-bsml-fixedbar bsml底部转换工具组件

标题|内容
----|----
类型|通用
支持布局|fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-bsml-fixedbar/mip-bsml-fixedbar.js

## 示例

### 基本用法
```html
<mip-fixed type="bottom">
    <mip-bsml-fixedbar>
        自定义内容，可以嵌套其他组件
    </mip-bsml-fixedbar>
</mip-fixed>
```

需要配合mip-fixed组件实现fixed定位，fixed位置参考mip-fixed使用方式
