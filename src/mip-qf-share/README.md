# mip-qf-share

mip-qf-share 湖南七风119手游网游戏分享组件

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-qf-share/mip-qf-share.js

## 示例

### 基本用法
```html
<mip-qf-share qrcode-url="https://m.119you.com/hadg0/">
    <br/>分享
</mip-qf-share>

```

## 属性

### qrcode-url

说明：微信分享二维码
必选项：否
类型：URL类型
默认值：当前页面的URL
