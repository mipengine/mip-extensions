# mip-qz-nav

前瞻网导航组件

描述|网站导航组件
----|----
类型|定制,业务
支持布局| responsive,fixed-height,fill,container,fixed
所需脚本|https://mipcache.bdstatic.com/static/latest/mip-qz-nav.js

## 属性

### rawurl
说明：当前页面的rawurl 如果rawurl包含li a的href属性 li添加class="cur"
必填：是
格式：字符
取值：字符串
默认值："/"
