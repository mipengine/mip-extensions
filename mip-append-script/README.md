# mip-script-url

mip-script-url用于添加第三方js的js脚本代码

|描述|脚本mip改造|
|---|---|
|类型|脚本|
|支持布局|N/S|
|所需脚本|https://mipcache.bdstatic.com/static/v1/mip-append-script/mip-script-url.js|

## 示例


```html
<mip-script-url url="https://apps.bdimg.com/libs/jquery/1.8.2/jquery.min.js"></mip-script-url>
```

## 属性

### url

说明：需要添加的js的url
必选项：是
类型：字符串
